# To-Do App

This project was built as a way to study the basics of frontend development with HTML, CSS, and JavaScript.

My main goals with this project are:

- Practice the core structure of a frontend application
- Learn better habits when working with functions
- Improve how I organize code and separate responsibilities
- Understand how small interactive features work in the browser

This is also a learning record. As I keep reading through the code I wrote, I plan to add comments to the files explaining what I learned and why each part works the way it does.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks in `localStorage`

## Project files

- `index.html` for the page structure
- `style.css` for the layout and styling
- `app.js` for the application logic

## Run the project

Open `index.html` in your browser.
